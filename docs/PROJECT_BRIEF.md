## About Lunar Core

**Lunar Core** is a WordPress plugin that provides the foundation for documentation websites through Custom Post Types, Taxonomies, Gutenberg Blocks, and various supporting features.

The plugin focuses on data management and website functionality, ensuring that content remains structured, maintainable, and sustainable over the long term.

---

## Project Goals

Lunar Core is developed to:

- provide a stable foundation for WordPress-based documentation websites;
- leverage native WordPress features whenever possible;
- provide Gutenberg Blocks specifically designed for documentation;
- maintain a simple, consistent, and maintainable data structure.

---

## Scope

Lunar Core includes the development of:

- Custom Post Types
- Taxonomies
- Gutenberg Blocks
- Metadata
- Shared Services
- WordPress integrations

Visual presentation, layout, and styling are outside the scope of the plugin.
